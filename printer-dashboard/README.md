# لوحة تحكم الطابعات (Printer Dashboard)

لوحة ويب بـ Python (Flask) تتحكم في طابعاتك من مكان واحد:
- **Bambu Lab** عبر LAN mode (تتبع + طباعة كامل)
- **Snapmaker U1** عبر LAN mode (إرسال طباعة، تتبع محدود)

## التشغيل

```bash
cd printer-dashboard
pip install -r requirements.txt
python server.py
```

ثم افتح: http://localhost:8080

## الإعداد

عدّل `config.py` واكتب بيانات طابعاتك:
- Bambu: `ip` + `access_code` (من إعدادات الطابعة → LAN mode)
- Snapmaker U1: `ip` + `port` (منفذ LAN)

## الاستخدام

1. اكتب مسار ملف G-code في الخانة.
2. اضغط **اطبع على الكل** لإرسال الأمر لكل الطابعات،
   أو اضغط **اطبع على هذه** لطابعة محددة.
3. الحالة تتحدث تلقائياً كل 5 ثواني.

## ملاحظة مهمة لـ Snapmaker U1

بروتوكول LAN الخاص بـ U1 غير موثق بالكامل من Snapmaker،
فالتتبع المباشر (حرارة/تقدم) محدود. إن فشل إرسال الطباعة عبر المنفذ،
ارفع الملف يدوياً عبر تطبيق Luban، وشارك نتيجة `nmap` على IP الطابعة
عشان نكتشف المنفذ والبروتوكول الصحيح ونكمل الربط البرمجي.
